# BRM-Cadence

이 데이터필드는 케이던스와 평균 또는 최대 케이던스를 표시합니다.

평균 케이던스와 최대 케이던스를 선택하여 표시할 수 있습니다.
기본 색상테마는 빠른 케이던스에서 초록색, 느린 케이던스에서 빨간색으로 표시됩니다.
색상은 설정에서 변경 가능합니다.
개인목표 케이던스를 지정하면 해당 케이던스와 비교하여 색상으로 확인할 수 있습니다.

가민 익스프레스 또는 가민커넥트 앱에서 설정을 변경할 수 있습니다.
